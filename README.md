# How to build


```js
sudo yarn install
```

## for Chinese users

```js
npm install -g cyarn --registry=https://registry.npm.taobao.org

npm config set puppeteer_download_host=https://npm.taobao.org/mirrors

cyarn install
```


# How to test

```js
npm run build

npm run test:watch
```
